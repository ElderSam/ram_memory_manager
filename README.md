# App to manage RAM memory usage

- memory consumption viewing 
- update info every second

---------

## Run;
`npm run dev`

## Stats
accessing the route [http://localhost:3000/stats](http://localhost:3000/stats)

Example:

```json
{
  "stats": {
    "OS": "win32",
    "Arch": "x64",
    "TotalRAM": "6020 MB",
    "FreeRAM": "777 MB",
    "Usage": "87.09 %"
  }
}
```
